# WLAN-RELAIS

This project is a WLAN-RELAIS for the Raspberry Pi Pico W. It can be used to control a relay via WLAN. The relay can be used to switch on/off a device. The project is based on the pico-sdk and the pico-examples. The project is written in C++ and uses the pico-sdk for the hardware access.


Hardware:
- Raspberry Pi Pico W
- OLED Display (I2C)
- Relay Module (GPIO)
- RGB LED based on WS2812B (GPIO)
